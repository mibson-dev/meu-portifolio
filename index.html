Quem sou

Sou desenvolvedor fullstack da Bahia, cursando Análise e Desenvolvimento de Sistemas. Tenho foco em backend Java com Spring Boot, mas construo o ciclo completo: da API ao frontend, do desenvolvimento ao deploy em produção.

Trabalho com arquitetura em camadas, APIs REST com autenticação JWT, controle de acesso por perfis, integração com PostgreSQL e boas práticas como tratamento de exceções e validações robustas.

Já entreguei projetos reais com regras de negócio complexas, hospedados em produção e acessíveis publicamente.
